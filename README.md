# 便利屋かんたん概算見積もりシミュレーター

## 構成ファイル

| ファイル | 説明 |
|---------|------|
| `index.html` | お客様向け見積もりシミュレーター画面 |
| `admin.html` | 管理者向け料金管理画面 |
| `style.css` | 共通スタイルシート |
| `data.js` | 料金データ・サービスデータ定義 |
| `app.js` | 見積もり計算ロジック |

---

## サイトへの埋め込み方法

### 方法1：iframeで埋め込む（最も簡単）
```html
<iframe src="https://あなたのドメイン/benriya-simulator/index.html" 
  width="100%" height="1000" frameborder="0"
  style="border:none; border-radius:12px;">
</iframe>
```

### 方法2：同じサーバーに設置
`index.html` `admin.html` `style.css` `data.js` `app.js` を  
同じフォルダにアップロードしてください。

## 管理画面へのアクセス

URL: `admin.html` を開く

初期ログイン情報:
- メール: `admin@benriya.com`
- パスワード: `admin123`

**※本番運用前にパスワードを変更してください。**  
`admin.html` の先頭にある `ADMIN_PASS` の値を変更してください。

---

## 料金の変更方法

1. `admin.html` を開いてログイン
2. 上部ナビの「料金ルール」をクリック
3. 変更したいサービスのボタンをクリック
4. 料金を編集
5. 「変更を保存する」ボタンをクリック

料金データは **ブラウザのlocalStorage** に保存されます。

---

## LINE・電話・フォームの設定方法

### LINE
`admin.html` → 「全体設定」タブ → LINE公式アカウントURLを設定  
または `app.js` の `buildLineMessage` 関数を修正

### 電話番号
`index.html` と `admin.html` の電話番号（`0280-XX-XXXX`）を実際の番号に変更

### フォーム送信
`app.js` の `submitForm()` 関数内に実際のAPI呼び出しを実装してください。

## 料金計算ロジックの説明

### 基本構造
```
概算最低額 = 基準金額 × 最小倍率（デフォルト1.0）
概算最高額 = 基準金額 × 最大倍率（デフォルト1.3〜1.5）
```

### サービス別計算例

**草刈り**
```
基準金額 = 基本料金(8,000) + 面積×200円 + 草高さ加算 + ゴミ処分 + 作業条件
```

**木の伐採**
```
基準金額 = 基本料金(10,000) + 高さ別料金×本数 + 幹太さ + 処分費 + 高所費
```

**不用品回収**
```
基準金額 = 量別基本料金 + 重量物加算 + 階段加算 + 分別費
```
